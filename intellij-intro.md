# Intellij Mac OSX 단축키
커맨드 + E => 최근에 본 파일

커맨드 + 쉬프트 + E => 최근에 수정한 것

커맨드 + 쉬프트 + O => 모든 파일을 찾는 것

커맨드 + O => 클래스 파일만 찾는 것

커맨드 + 쉬프트 + F => Find path ( 전체 검색하는 것 )

커맨드 + K => 커밋

커맨드 + F10 => Update 프로젝트

옵션 + F12 => 터미널 열기

커맨드 + R => replace (match, regex, etc ... )
